# < Foxy Framework (C++) >
 Experimental C++ app framework.

## - Requirements -
 Built with CMake 3.24, C++23, Visual C++ compiler, and Visual Studio 2022 generator.
 
## - Dependencies -
 - GLFW
 - GLM
 - spdlog - Custom fork
 - BS Thread Pool
 - Vulkan: ***"Batteries not included!"*** Separate install required.
